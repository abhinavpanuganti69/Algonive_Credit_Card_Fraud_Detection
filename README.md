# 💳 Credit Card Fraud Detection System

## Overview

This project implements a **Credit Card Fraud Detection System** using Machine Learning in **Google Colab**.
The model analyzes transaction data and classifies them as **fraudulent** or **legitimate** based on patterns.

---

## Features

* Implemented entirely in Google Colab
* Data preprocessing and normalization
* Handling imbalanced dataset
* Exploratory Data Analysis (EDA)
* Model training and evaluation
* Fraud detection using classification algorithms

---

## Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* Google Colab

---

## Dataset

* Credit Card Fraud Detection dataset
* Contains anonymized features (V1–V28)
* `Class` column:

  * 0 → Legitimate
  * 1 → Fraud

---

## How to Run (Google Colab)

1. Open Google Colab
2. Upload the notebook file (`.ipynb`)
3. Upload the dataset (CSV file)
4. Run all cells step-by-step

---

## Workflow

1. Import libraries
2. Load dataset
3. Data preprocessing
4. Handle class imbalance
5. Train model (e.g., Logistic Regression)
6. Evaluate model performance

---

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score

## Results

The model is optimized to detect fraudulent transactions with high recall, minimizing false negatives.
